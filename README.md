
# Projeto CRUD

Projeto feito para estudar CRUD através de um cadastro de clientes utilizando localStore como banco de dados.



## Funcionalidades

- Cadastrar novo cliente
- Editar cliente já existente
- Excluir cliente 
- Responsivo
